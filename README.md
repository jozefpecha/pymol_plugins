# pymol_plugins
Repository for short and useful pymol plugins

## Instalation
Install specific plugin by copying its permalink into PyMOL Plugin Manager.
Visit https://pymolwiki.org/index.php/Git_install_scripts for installing the whole repository.

## Usage
**visualize_pharms:** 
```
pharm_view pharmacophore_name, mol_name, sphere_scale, sphere_transparency
```
`pharmacophore_name` and `mol_name` are required names of loaded objects. `sphere_scale` and `sphere_transparency` are optional arguments that default to 0.35 and 0.5 respectively.
